# Support

HookShip is in pre-release development. The public
[`toolkit`](https://github.com/HookShip/toolkit) and
[`hook-service`](https://github.com/HookShip/hook-service) repositories are
under active development, and no public packages or releases are available.

For community help:

1. Read the affected repository's documentation.
2. Search its existing issues.
3. Open an issue using the appropriate template with reproducible details.

Use the repository where the relevant code or documentation belongs; see the
[repository placement policy](https://github.com/HookShip/.github/blob/main/REPOSITORY_PLACEMENT.md).
Repository-specific support guidance overrides this shared policy.

Community support is best effort. There is no response-time or resolution-time
SLA. Do not include secrets, credentials, private payloads, or personal data.
For vulnerabilities, follow the [security policy](https://github.com/HookShip/.github/blob/main/SECURITY.md)
instead of opening a public issue.
