# Code of Conduct

## Our standard

HookShip aims to provide a respectful, inclusive, and constructive environment.
Participants should:

- communicate with empathy and technical clarity;
- welcome differing experience levels and viewpoints;
- give actionable feedback focused on the work;
- respect privacy, boundaries, and attribution; and
- accept responsibility for mistakes and help repair their impact.

Harassment, discrimination, threats, sexualized attention, deliberate
intimidation, doxxing, and sustained disruptive behavior are not acceptable.

## Scope

This code applies in HookShip repositories and other spaces where someone is
representing the organization.

## Reporting and enforcement

Report conduct concerns privately to the maintainers using an available private
GitHub communication channel. If no private channel is available, open a
minimal public issue asking a maintainer to establish private contact; do not
include sensitive details.

Maintainers will review reports in context and may edit or remove content,
limit participation, or restrict access when necessary. Reports will be handled
as discreetly as practical, but confidentiality cannot be guaranteed.
