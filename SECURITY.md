# Security Policy

HookShip is in pre-release development. No public release or operated hosted
service is currently available.

## Reporting a vulnerability

Use **GitHub private vulnerability reporting in the affected repository**:

1. Open the repository's **Security** tab.
2. Choose **Report a vulnerability**.
3. Include the affected component or revision, impact, reproduction steps, and
   any suggested mitigation.

Do not disclose vulnerability details in a public issue, pull request,
discussion, or commit.

If private vulnerability reporting is unavailable, open a minimal public issue
in the affected repository asking maintainers to enable or provide a private
reporting path. Do not include vulnerability details in that request.

Reports are evaluated on a best-effort basis. Because the project is
pre-release, response or remediation timelines are not guaranteed.
